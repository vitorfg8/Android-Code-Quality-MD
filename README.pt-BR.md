# Regras de Qualidade de Código Android para Assistentes de IA

🇺🇸 English version: [README.md](README.md)

Um padrão reutilizável de qualidade de código para Android/Kotlin com foco em:

- Compatibilidade com Detekt
- Princípios SOLID
- Boas práticas de Kotlin
- Segurança com Coroutines
- Convenções de recursos Android
- Localização e internacionalização
- Guidelines de API e performance do Jetpack Compose
- Legibilidade e manutenibilidade
- Geração de código pronta para produção usando IA
